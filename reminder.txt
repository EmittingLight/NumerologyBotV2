Памятка по использованию бота-калькулятора в Telegram

1. Запуск бота

Для начала работы с ботом отправьте команду /start.
Бот приветствует вас и предлагает выбрать одно из действий, используя кнопки в интерактивном меню.

2. Регистрация

Нажмите кнопку "Зарегистрироваться в боте", если требуется регистрация для доступа к функционалу.

3. Использование калькулятора

Нажмите кнопку "Запуск калькулятора" для начала процедуры расчета.
Сначала выберите день рождения из предложенных опций.
Затем выберите месяц рождения.
Введите год рождения в формате YYYY (например, 1987).

4. Получение результатов

После ввода всех данных бот создает и отправляет изображение с эзотерическим расчетом.
Бот также предлагает кнопки для дальнейших действий, например, ввод новой даты или получение подробной расшифровки.

5. Дополнительные опции

В любой момент можно запросить "Памятку по работе с ботом" для получения справочной информации.
Если возникнут вопросы или проблемы, используйте кнопку "Обращение в службу поддержки".

6. Важные детали

Все коммуникации с ботом происходят через текстовые сообщения и кнопки в интерактивном меню.
При возникновении ошибок бот уведомит вас и предложит повторить попытку.
Эта памятка поможет вам эффективно использовать бот-калькулятор для расчетов по методике "Теневая Матрица Судьбы".
